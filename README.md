### Course Description:

This class will introduce you to principles of programming using the Python programming language.  We will attempt to cover the following topics in some depth:

+ network programming
+ functional programming
+ systems programming
+ object-oriented design


#### Pre-Requisites:

+ Prior programming experience is recommended, but not required

**Start Date**: Thursday, April 2, 2015

**Time & Duration**: Thursdays, 7-9pm. 8 weeks.
